# 右+左 連射ツール

右クリックを押しながら左クリックしている間だけ、左クリックを連射するWindows用ツールです。

## 普段の起動

開くものはこれです。

`起動はこちら.lnk`

デスクトップにも `起動はこちら - 右+左 連射ツール.lnk` を作成済みです。

タスクバーにピン留めする場合は、古いピン留めを外してから、このショートカットを右クリックしてピン留めしてください。

## 配布

配布するファイルはこれ1つです。

`配布用\右+左 連射ツール.exe`

このexeはPython不要で起動できます。

## 編集

VS Codeで編集するときは、このフォルダだけを開きます。

`編集はこちら.lnk`

デスクトップにも `編集はこちら - 右+左 連射ツール.lnk` を作成済みです。

またはVS Codeの「フォルダーを開く」で `C:\Users\tkt01\rensha-tool` を選びます。

## 使い方

1. `起動はこちら.lnk` を開きます。
2. 必要なら「連射速度」を変えます。
3. 対象画面で右クリックを押したまま、左クリックします。
4. 左クリックか右クリックを離すと連射が止まります。

## メモ

- Windows専用です。
- 右クリック自体も対象アプリに届きます。
- 使用先のルールや規約に従ってください。
