# 右+左 連射ツール

右クリックを押しながら左クリックしている間だけ、左クリックを連射するWindows用ツールです。

## 起動

配布用フォルダのexeを開きます。

```text
配布用\右+左 連射ツール.exe
```

Pythonは不要です。

## 配布

配布するファイルはこれ1つです。

```text
配布用\右+左 連射ツール.exe
```

## 編集

ソースコードはこれです。

```text
rensha_tool.py
```

VS Codeで編集するときは、フォルダーとして `C:\Users\tkt01\rensha-tool` を開いてください。

## 使い方

1. `配布用\右+左 連射ツール.exe` を開きます。
2. 必要なら「連射速度」を変えます。
3. 対象画面で右クリックを押したまま、左クリックします。
4. 左クリックか右クリックを離すと連射が止まります。

## フォルダ構成

```text
rensha-tool
├─ README.md
├─ rensha_tool.py
├─ .gitignore
└─ 配布用
   └─ 右+左 連射ツール.exe
```

## メモ

- Windows専用です。
- 右クリック自体も対象アプリに届きます。
- 使用先のルールや規約に従ってください。
